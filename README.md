### 背景
> 因为放烟花会污染空气，现在过年在家都看不到漂亮的烟花了，这次我们在曈星阁里放烟花，没有污染的那种哦，大家可以在烟花里面写祝福……快去点亮烟花看看我给你写的祝福吧！
